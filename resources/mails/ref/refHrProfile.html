На прошлой неделе закончилась 10-я стажировка Java Enterprise, в базе появились свежие вакансии.

И появилась возможность из своего профиля выбирать кандидатов, добавивших резюме с определенной даты (выбор даты и ссылка на базу вверху профиля). Также Вы можете заполнить пустые поля профиля для лучшего сотрудничества.

Возможно также Вас заинтересует рефферальная программа бонусов (заранее извиняюсь, если нет). На каждую регистрацию (получение первого бесплатного занятия) по реферальным ссылкам Вам будет приходить уведомительное письмо и при оплате добавляться бонусы, которые можно будет перевести в деньги (подробности по запросу).

Спасибо за сотрудничество. Надеюсь, что Вы найдете в базе кандидатов достойных Java разработчиков.
