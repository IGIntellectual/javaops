Для персональной регистрации IntelliJ IDEA на полгода:
  • зайди по этой ссылке
  • выбери в New Subscription кнопку BUY NOW
  • следуй этой инструкции (с пункта 2)
  • под Product IntelliJ IDEA есть линк "Have a discount code?", введи туда купон
  • после завершения заказа на почту приходит письмо с сертификатом. В IDEA я его активировал через Help -> Register...-> JetBrains Account -> email/password -> Activate.
    Возможно также активировать по ссылке из письма и "Download activation code for offline usage" или через "Legacy license key"
Не спеши при регистрации. Если c нескольких раз не получилось, пиши в личку skype или slack.
