16.03 старт 10-го выпуска проекта-стажировки Java Enterprise (Topjava)

Разработка полнофункционального Spring/JPA Enterprise приложения c авторизацией и правами доступа на основе ролей с использованием наиболее популярных инструментов и технологий Java: Maven, Spring MVC, Security, JPA(Hibernate), REST(Jackson), Bootstrap (css,js), datatables, jQuery + plugins, Java 8 Stream and Time API

Проект постоянно развивается, в том числе благодаря вашему активному участию, появляются новые отзывы участников, новые истории успеха, новые технологии (Bootstrap Glyphicons, XSS, JPA 2.1 EntityGraph, Jackson @JsonView, методы улучшения качества кода) , новый дизайн а также выпускной проект, который каждый участник пишет самостоятельно (взят из реального тестового задания).

    Основное, что можно получить от участия в проекте:
  • понимания технологий, как работают фреймворки, на чем они основаны, какие паттерны используются
  • навыки нахождения правильных практических решений для конкретных задач и решений конкретных проблем, из которых состоит ежедневная работа программиста (~75% это умение выбрать из найденного на Stackoverflow)
  • работа с инструментами: Git, IDEA, Maven, Tomcat, DB клиент, тестирование REST сервисов
  • привычка кодировать красиво
  • кругозор, навыки решения подобных задач
Мы учимся мыслить и работать как Java разработчики, на работе это будет гораздо сложнее и стоить дороже. Вот на мой взгляд хорошие советы новичкам. От себя я добавлю:
  • Учись грамотно формулировать проблему. Проблема "у меня не работает" может иметь тысячи причин. В процессе формулирования очень часто приходит ее решение.
  • Учись исследовать проблему. Внимательное чтение логов и умение дебажить - основные навыки разработчика. В логах надо читать верх самого нижнего эксепшена - там причина всей портянки.
  • Грамотно уделяй время каждой проблеме. Две крайности - сразу бросаться в чат или ко мне за помощью и бится нам ней часами. Пробуй решить ее сам и в зависимости от проблемы выделяй на это разумное время.

Стажировка реально дает возможность устроится на позицию middle разработчика (особенно при выполнении всех Домашних Заданий).

Требования к участникам:

На проекте мы не проходим синтаксис java: предполагается, что вы уже знаете, что такое классы, конструкторы, аннотации, анонимные классы, исключения и пр. Также необходим некоторый опыт программирования на java. Желательно знание сервлетов. Мы начинаем использовать их на первом занятии, кратко повторяя основы. Знание SQL (на уровне Select/ Insert/ Update/ Delete), JavaScript желательно, но не принципиально. Весь "Enterprise": Maven, Git (в том объеме, который нам нужен), Spring, Hibernate, REST даются с азов. Никаких специальных знаний Java EE не требуется.
