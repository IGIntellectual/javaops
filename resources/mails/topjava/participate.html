Общение на проекте предлагаю на "ты". Проект ведется по подготовленным на основе вебинара, разбитым по темам и отредактированым записям. Все домашние задания являются чаcтью проекта, подобранны по темам занятий и требуют реализации функциональности, схожей с пройденной на занятии.

IDE: JetBrains IDEA

IDEA считается лучшей Java IDE, и по последней статистике IntelliJ is the number one IDE.
Поэтому разработка проекта ведется в JetBrains IDEA, брать версию Ultimate
Если ты студент (или есть знакомые студенты/ аспиранты), то лучше взять студенческую учебную лицензию JetBrains IDEA на год. Купон на бесплатную личную лизензию IntelliJ IDEA на полгода можно получить согласно политике JetBrains только один раз по этой ссылке и далее по инструкции в письме (лучше активировать ее после месячного trial периода, купон будет действителен).

Проверка домашних заданий

Разбор домашний заданий происходит в начале каждого занятия.
Если у тебя участие с проверкой ДЗ, то свое решение вступительного задания желательно присылать на проверку начиная c 02.12 до вечера 06.12, остальных домашних заданий - до вечера вторника.

Проверка заданий подразумевает ревью твоего кода. Ты увидишь именно свои ошибки в стиле, структурах данных, алгоритмах и кодировании. Это самый эффективный способ научиться программировать. Все домашние задания это часть проекта. Часть из них достаточно простая, часть Optional - сложнее. Обычно участники чекинят код в гитхаб и шлют мне ссылку на репозиторий. Я проверяю и фидбэк пишу в slack. Получается эффективно и оперативно. Можно что-то переделать и снова поревьюится.

Также для всех участников: бесплатное (или с доплатой за проверку) участие в следующий выпусках проекта Topjava (ближе к концу проекта я вышлю ссылку). Проект постоянно развивается, в том числе благодаря вашему активному участию. По сравнению с первыми выпусками проект существенно обновился.
