 Онлайн обучение Java Core/ Web/ Enterprise.
Совместная разработка проектов. Стажировки и помощь в трудоустройстве.

Уникальное, быстрое практическое освоение различных технологий и фреймворков Java за счет совместного создания проекта, решения практических задач в ходе его разработки (все домашние задания являются частью одного проекта), ревью кода и обратной связи.
  • Если ты новичек, это твой шанс освоить новую профессию на отлично, без долгих блужданий по просторам интернета
  • Если в поиске работы Java разработчика, после стажировки шансы получить хорошую работу возрастут на порядок
  • Если уже работаешь на Java, но мало опыта в Enterprise, стажировка это отличный способ получить его быстро и на хорошем уровне

Истории/Отзывы

Последовательная возрастание сложности

Сложность проекта возрастает постепенно, с кода "Hello!" в main() до финальной версии в Heroku. Слон съедается по кусочкам.

Общение, поддержка группы и преподавателя

Общение в группе проекта Slack по любым вопросам (дополнительно к занятиям): git, idea, трудоустройство, новости IT и пр.
В группе выпускников обсуждение вступительных заданий, свежие вакансии, цены на рынке труда, события IT, помощь, интересные видео...

Доступность материалов

Все видео доступно для просмотра онлайн и скачивания (в случае вебинаров - сразу после занятия). Записи разбиты по темам и тщательно отредактированы. Большое количество подобранных ресурсов для углубления в детали.

Ревью Вашего кода

Вы увидите именно свои ошибки в стиле, структурах данных, алгоритмах, кодировании.
Опыт равноценен нескольким годам работы Java разработчиком.

Совместная разработка

Участники на протяжении всего курса работают с кодом проекта. Выполнение домашних заданий закрепляет полученные знания. В результате проект становится "Вашим".

Оптимальное подобранное время

Занятия проходят раз в неделю на стажировках Topjava и Masterjava, время необходимое для проработки пройденных тем и выполнения домашнего задания. Индивидуальный график обучения основ на Basejava.

Позиционирование в резюме стажировки Java Enterprise (Topjava)

  • Обязательно убери из резюме любое упоминание Junior. Количество обращений возрастет на порядок.
  • После завершения проекта ты освоишь все заявленные в нем технологии - вставь их в квалификацию.
  • В разделе опыт работы (если нет коммерческого опыта) вставь:

    Участие в разработке Spring/JPA Enterprise приложения c авторизацией и правами доступа на основе ролей на стеке:
    Maven, Spring MVC, Spring Security, REST (Jackson), Java 8 Stream API:

    • реализация сохранения в базы Postgres и HSQLDB на основе Spring JBDC, JPA(Hibernate) и Spring-Data-JPA
    • реализация и тестирование REST и AJAX контроллеров (более 100 JUnit тестов)
    • реализация клиента на Bootstrap (css/js), datatables, jQuery + plugins.
    • собственная доработка проекта

    Делай упор не на обучение, а на участие в проекте. Выполнение домашних заданий это полноценное участие с написанием функционала по всем пройденным технологиям. На собеседовании смотрят не на то, что ты заканчивал, а на опыт и знания.

В процессе обучения

  • Если рассмотриваешь предложения по работе, подними в своем профиле этот флаг и обязательно заполни ссылку на резюме. Обновления нашей базы выпускников смотрят уже более 125 партнеров по трудоустройству (компании и индивидуальные рекрутеры). Проверь содержание "Информация для HR": по нему принимают решение, открывать резюме или нет.
  • Вступай в нашу группу участников Slack: каналы помощи с Java, отзывы о работодателях, обсуждение тестовых заданий, вакансии, цены на рынке труда, IT события, интересные видео и многое другое.
  • Подпишитесь на рассылку вакансий под себя
