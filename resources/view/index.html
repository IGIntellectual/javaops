  • Максимальный охват

    Самые важные и необходимые Java разработчику темы, API, фреймворки, инструменты в одном курсе.

  • Насыщенное и структурированное изложение как дополнение к проекту

    Изложение материала "без воды" нацелено на практическое использование и дополняется материалом для самостоятельной проработки. Темы идут не по отдельности (в воздухе) на отстраненных примерах, а последовательно, в связке, как дополнения к созданию единого работающего приложения.
    Усвоение и понимание материала увеличивается в разы.

  • Последовательная возрастание сложности с нуля

    Сложность проекта возрастает постепенно, с кода "Hello Topjava" в main() вступительного задания до финальной версии. Слон съедается по кусочкам..

  • Совместная разработка

    Участники на протяжении всего курса работают с кодом проекта. Выполнение домашних заданий закрепляет полученные знания.
    В результате проект становится "Вашим".

  • Общение, поддержка группы и преподавателя

    По любым вопросам: git, трудоустройство, зарплаты программиста, java консультации, архитектура, новости IT и пр.

  • Доступность и структурированность материалов

    Все видео доступно для просмотра онлайн и скачивания (в случае вебинаров- сразу после занятия). Записи разбиты по темам и тщательно отредактированы. Большое количество подобранных ресурсов для углубления в детали.

  • Оптимальное подобранное время

    Занятия проходят раз в неделю, время необходимое для проработки пройденных тем и выполнения домашнего задания

  • Ревью Вашего кода.

    Вы увидите именно свои ошибки в стиле, структурах данных, алгоритмах, кодировании.
    Опыт равноценен нескольким годам работы разработчиком java.
