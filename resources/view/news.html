Действует бонусная программа при оплате проекта новым участником, пришедшим по личной реферальной ссылке (скопиовать из профиля).
В профиле стоимость на проекты рассчитывается автоматически с учетом бонусов и участия в других проектах.

Переработаны почти все занятия обновленного проекта Разработка Web приложения "База данных резюме" (от массивов до CRUD веб приложения). Скоро ожидаются первые выпускники.

Проект идет в индивидуальном режиме (старт в любое время) по готовым материалам с поддержкой в мессенджере Slack и личным куратором. Все домашние задания являются частью веб проекта, их разбор идет в начале каждого занятия.
Так будет выглядеть мое резюме в разрабатываемом на этом проекте приложении.

Планируется интеграция с сервисом поиска разработчиков GetCoder.

Пожалуйста, заполните у себя в профиле новое поле GitHub account (или достаточно залогиниться через GitHub).
Не забудьте указать в прфиле GitHub локацию - это важная информация для HR.

Группу Slack JavаOPs стала доступной всем участникам (не только выпускникам Topjava)

В группе есть каналы помощи с Java, отзывы о работодателях, обсуждение тестовых заданий, вакансии, цены на рынке труда, IT события, интересные видео по теме и многое другое. Вступить в группу можно из Личного профиля. Присоединяйтесь, если еще не с нами!
